# Clippy

Clipboard manager with RPC and Talon support. Supports copying and pasting multiple things at once.

## Examples

-   `copy air`
-   `paste air`
-   `chuck air`

### Multiple targets

-   `copy air and bat`
-   `copy air past bat`
-   `copy 2 items air`
