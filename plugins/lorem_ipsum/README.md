# Lorem ipsum generator

## Usage

`"lorem ipsum"` generates 30 words of lorem ipsum.

`"lorem ipsum <number>"` generates `<number>` words of lorem ipsum.

### Example

`"lorem ipsum ten"` => `Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do.`
